# Mars Weather Terminal 

Консольное приложение для мониторинга погоды на Марсе с использованием NASA InSight API.

## Функционал
* Получение свежих данных о температуре, ветре и давлении.
* Строгий "терминальный" интерфейс вывода.
* Автоматическое логирование данных в CSV-файл.
* Безопасное хранение API-ключа через переменные окружения.

## Установка
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install requests python-dotenv`
3. Создайте файл `.env` и добавьте свой ключ: `NASA_API_KEY=ваш_ключ`
4. Запустите: `python Main.py`
